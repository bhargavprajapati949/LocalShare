  • • Path-safety boundaries across shared root directories

🍎 Mac: Quick Install

Apple blocks unsigned indie apps downloaded from a browser. For a 1-click installation that bypasses this warning and puts LocalShare directly on your Desktop, open Terminal and paste this command:

FAQ

Does LocalShare require internet access?

No. It is designed for local-network sharing between devices on the same LAN/hotspot.

Can I protect access with a PIN?

Yes. You can enable an optional session PIN and enforce runtime permission policies from admin UI.

How do users get the newest version?

The download buttons use GitHub latest-release URLs, so each click resolves to your newest release asset.
